1%GETXSTAR Returns the xStar value given either the height of the wheel contact
 2%point relative to the pan coordinate frame and either the beta or gamma
 3%joint value. It is assumed that the angle input represents the beta joint
 4%angle if selector = false, and the angle input is the gamma joint angle if
 5%selector = true
 6%
 7%Inputs:
 8%-z: The height from the pan coordinate frame to the wheel contact point.
 9%-angle: The angular value that is either beta or gamma, depending on the
10%selector input.
11%-selector: A logical that indicates that the angle value represents beta
12%-kC: A struct containing the kinematic constants of the Sherpa TT leg.
13%if selector=true, or gamma if selector=false.
14%Outputs:
15%-xStar: The radius in a cylindrical coordinate representation that
16%connects the pan coordinate frame to the wheel contact coordinate frame.
17%
18%sherpaTTFK.m
19%author: wreid
20%date: 20150216
21
22function xStar = getXStar(z,angle,selector,kC)
23
24    A = kC.l2+kC.l4*cos(kC.zeta)-kC.l7;
25    B = kC.l1-kC.l4*sin(kC.zeta)-kC.l6-kC.l8-kC.r;
26
27    if ~selector
28        xStar = A+kC.l3*cos(asin((-z+B-kC.l5*sin(kC.zeta+angle))/kC.l3))+kC.l5*cos(kC.zeta+angle);
29    else
30        xStar = A+kC.l3*cos(angle)+kC.l5*cos(asin((B-kC.l3*sin(angle)-z)/kC.l5));
31    end
32
33end
